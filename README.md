# ISM2411_M13

## WEEKLY WORKS HOUR TRACKER!

Welcome to the weekly work hours tracker. The purpose of this tracker is for people to track how many hours they are on track to work in a given seven day (week) period.

## To run this program, use the following commands:
python app.py
python3 app.py

## YOUR OUTPUT SHOULD LOOK LIKE THIS
#### Example: 10 Hours

How many hours were you working for today? 5
Based off your input, if you work the same hours daily (for 7 days), you are on track to work 70.00 hours this week.
